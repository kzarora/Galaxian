Bomb Attack


Ranvijay and Shyam are brave Commanding Officers in INDO CHINA War . They have spotted an intruder who is planting bombs. They sent message to the commanding headquarter containing characters W and B where W represents a wall and B represents a Bomb. They asked headquarter to tell them how many walls will be destroyed if all bombs explode at once.
*Note One bomb can destroy 2 walls on both sides.

Input:
First line of input contains number of test cases T.
Each test case contains a single string which contains two type of characters 'W' and 'B'.

Output:
For each test case print the total number of destroyed wall.

Constraints:
1 ≤ T ≤ 10
1 ≤ |S| ≤ 10

Example:
WWBW
BWWWBWBWW

Output
3
6
