Matrix Game


Given a NxM matrix of integers. Find a number 'x' in the matrix if number found in the matrix then replace the all element that column and row with the number 'x'.

Input
A is the number test cases.
Each test case
Three space separated numbers N,M and X.
Followed by N lines each lines consists M space separated numbers.

Output
NxM matrix after performing given operation.

Example
Input
1
3 3 5
1 2 3
4 5 6
7 8 9

Output
1 5 3
5 5 5
7 5 9
