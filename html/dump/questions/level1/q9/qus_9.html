Color The Ball



There are N balls numbered 1 to N and colored white. N students came and play a coloring game first student come paintthe balls 1,2,3,4.........N with color black, 2nd student come and paint the balls 2,4,6,8.......(Coloring the ball white to black or black to white). This process goes on till Nth student, print the all ball numbers which color is black.

Input:
A the number of test cases.
Followed by one line for each test case contains a number N(number of students).

Output:
One line for each test case that contains the numbers on ball specified above.

Constrain:
1<=A<=1000

Example:
Input
1
5
Output
1,4
