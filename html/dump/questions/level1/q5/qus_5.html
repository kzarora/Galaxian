The Love One

Neeraj is a beginner in computer studies, he is good in judging patters and also very enthusiastic and thus try new things in what he studies, while studying Number system he observed that all positive no starts with '1' in base 2, beside of traditional number systems he checked some other no system like base 3,4... and observed are the starting with '1' or not
you task is to help Neeraj in how many bases are there which have starting with 1

Input
First Line have Number A denoting no of test cases.
Each test case has a no N which is to be converted into different bases for observation.

Output
Print no of bases having started with '1', in case of Unlimited bases Print "UNCOUNTABLE"

Constraints
1<=A<=10000
1<=N<=1000000000

Example
Input:
1
5
Output:
4

Explaination
Base 2,3,4,5
