Volcanic Eruption

Voyage of space is not an easy deed, In search of life Astronaut Harmit was traversing universe in his space ship Udyan, he reached a weird planet having some clues of life but excess of temprature, there was a volcanic eruption emitting magma, At the bottom of volcano there were many cubical rocks adjacent to each other having length and width identical i.e. 1*1 they were differing only in dimension of height, due to eruption magma is filled between these rocks, your task is to help Captain Harmit find amount of magma stored in these rocks.

Input:
The first line of input contains an integer A denoting the number of test cases. For each A there will be an integer N denoting no of Rocks 0, in the following line there will be n space separated integers which are height N rocks.

Output:
Print the volume of magma stored.

Constraints:
1 <= A <=100000000
1 <= N <=100000000
0 <= Wall Height < 20000000

Example
Input
1
6
1 3 2 5 0 2

Output
3
