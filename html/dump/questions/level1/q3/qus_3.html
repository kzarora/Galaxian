Info Protection

Meanwhile the mission the there is some very secrete information to be reached to Captain Harmit, but some country which were not willing success of India's this space mission tries to add some noise in that Message, But our team was ready for this step. there is a secrete encryption technique shared between Udyan Team (Team on action) and ISRO the were sending a Message in form of a stream of Numeric characters along with a number denoting hits. The Team try to convert that Numeric Stream into a Palindrome of Maximum Numeric Value Character Stream. If this is possible than Message is noise free and will be decrypted into original Message else noisy and need re transmission.
It’s a situation of Urgency can you help Team Udyan in its sub task of finding is it noisy or not?

Input Format
First Line contains A no of Test case.
First Line of each Testcase have 2 Numbers N no of characters in a Stream, K i.e. the maximum Hits allowed
Second Line of Each Test case will have a Numeric Character Stream


Constraints
1 <= A <= 200
1 <= N <= 2000000
1 <= K <= 2000000

Output Format
Print the Maximum Possible Palindrome Stream and in new line "No Noise"
If not Possible Print -1 and in new line "Retransmit"

Example:
Input
1
5 3
12345

Output
94349
No Noise
