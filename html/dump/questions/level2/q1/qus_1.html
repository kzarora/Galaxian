Genius are not made in a day, and so was our Captain Harmit, since his childhood he was very interested in math and computers science in his school day he and his friend Pari use pass a messages to each other, to maintain privacy they used an encrypting algorithm.
Encryption Include a message that is converted into a bit stream using ASCII format of 8 bits and a hash key k.
Encrypting Algorithm was to increase size of message stream k times by 1 bit and left shifting it(or appending 0 as Right Bit) by 1 bit each of k iterations than XOR ing these K streams. Few year later Harmit wants to decode some old messages and remember those days can you Decrypt these messages for Harmit.

Input Format
First line contains A no of test cases.
The first line of each test case contains two integers N and K i.e. length of encrypted bit stream and hash key Resp.
The second line contains bit stream consist of ones and zeros.

Constraints
1 <= A <= 200
1 <= N <= 2000
1 <= K <= 100

Output Format
Decoded message and print text.
If ASCII value message is not in an of alphabhet of Capital Case than use 'value%26' character considerind A is at index '0'

Example
1
18 3
011111100111111111

Output:
HI
