Word Ludo



Kids loves to play games so Ravi, but he love to play game made by him and are only for wise peoples. One day while playing Ludo he thought why only not to make a game of forming his own words, so he write different words on different faces of dice. Now he take any random word and try making it. By combinations of these faces

For example:
Let faces value of 6 face dice are:
"it" , "combination", "i", "comb", "nations", "sit"
And a random word:
w=[ combinationsit ]
Possible solution are
combinations it
comb i nations it

Input:
First line contains t denoting no of testcases.
Each test case has 3 lines.
First line is denotes the number of faces in a die (n) of combination of alphabets.
Second line contains n face values of a n faced die.
Third line the New word Ravi wants to form.

Output:
In new line Print all possible strings in one line separated by () in single line;
else If no string possible Print "NONE".


Note: Make sure the strings are sorted in your result.

Constraints:
1<=T<=1000
1<=N<=50
0<=l<=20
0<=s<=10000

Example:
Input:
1
6
it combination i comb nations sit
combinationsit

Output:
(combinations it)(comb i nations it)
