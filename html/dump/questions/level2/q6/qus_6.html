BMI Calculator



Given weight and height of a person, calculate BMI(Body Mass Index). Weight is in Kg and height is in feet and inches (Use 1 inch=2.54cm and BMI=weight/(height)^2 weight should be in Kg and height should be in meter).

INPUT:
 First line consists of a number T(number of test cases)
Each test case consists of two space seperated value first is weight and second is height(5'7" height input format).

OUTPUT:
Each line consists BMI of corresponding test case.


CONSTRAINTS:
3' to 8' height
25 to 125 kg weight
1<=T<=500

Example:
Input:
1
69 6'2.3"

Output:
19.373316
