Numbers



There are two given numbers X and Y, create a set of numbers using the digits of X which are greater than Y.

INPUT:
T is the number of test cases.
Followed by a line for each test case contains space separated two numbers X and Y.
OUTPUT:
For each test case a sigle line of output containing space separated numbers for corresponding input.

Constraint:
1<=T<=1000
1<=x,y<=999999999
Example:
Input:
2
6542 6400
89567 95800
Output:
6425,6452,6524,6542
95867,95876,96578,96587,96758,96785,96857,96875,97568,97586,97658,97685,97856,97865,98567,98576,98657,98675,98756,98765
